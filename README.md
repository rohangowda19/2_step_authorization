# 🔐 2-Step Authorization

A full-stack authentication system built from scratch with login, signup, and two-factor authentication (2FA) using plain HTML, CSS, JavaScript, Node.js, Express, and MongoDB.

---

## ✨ Features

- 📝 User Signup with name, email and password
- 🔑 Secure Login with hashed passwords
- 📧 Two-Factor Authentication (2FA) via OTP email
- ⏱️ OTP expires after 5 minutes for security
- 🔄 Resend OTP with 30 second cooldown
- ✅ Form validation with real-time error messages
- 🎨 Glassmorphism UI with pink-orange gradient theme
- 💾 MongoDB database to store users securely
- 🔒 Passwords hashed with bcryptjs — never stored in plain text

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript |
| Backend | Node.js, Express.js |
| Database | MongoDB, Mongoose |
| Email | Nodemailer + Gmail |
| Security | bcryptjs, OTP 2FA |
